GeneticAlg_SampleGen
====================

This application is just kept here for reference and in case it might become useful again.
It's not maintained and is not part of the unreal engine.



Simple tool to generate sample positions and mirror vectors for better SSAO
(uses genetic algorithm)

no legal issues in the sources
